Los Angeles County Public Library

The Public Library website was a complete site migration from their outdated site. I worked on the database architecture and implentation of the data to the front end for all 84 of the individual libraries.

I built a backend dashboard for L.A. County official librarians to properly and securely manage the data. I also setup a new book list system for their librarians to build out book lists to display on the site.

Link: https://lacountylibrary.org/

Code: Private

Los Angeles County Tax Collector

The LA County Treasurer and Tax Collector was another full site migration we handled at my current company. This site was filled with outdated information and applications and needed a complete makover.

I was tasked with rebuilding their core 15 form applications that make up this site. These forms range from applying for new business licenses, searching for business licenses, property tax applications, and many request for information forms.

Link: https://ttc.lacounty.gov/

Code: Private

Los Angeles County Digital Directory

This was definitely one of the most challenging projects I have ever worked on, and definitely the one I am most proud of. The Board of Supervisors for Los Angeles County tasked us with creating a digital directory for their 8 story building. I had to come up with a way to create an indoor wayfinding solution when I had never done so before. Seeing the final result was so rewarding and more succesful than I ever could have imagined.

I developed the entire wayfinding solution from front end to back end while providing my team with correct code guidelines to integrate the rest of the directory into the wayfinding map

Currently this project is not able to be linked out because it is secured by the Los Angeles County network. It only exist on our dev sites within their network and as a directory inside the actual building.

The Page Builder Plugin

This project is a plugin designed for WordPress to allow users to quickly mockup pages using a library of widgets. It gives you an instant visual representation of what you are building with full customization capabilities. This plugin allows users to edit and save widgets and page templates, then upload templates as pages. It centralizes all widgets and plugins into a dashboard where a user can edit a widget or template and change it across all pages that are associated with it. This is a monetary plugin birthed from RD Technology Solutions.

I have built this plugin entirely from bottom to top on my own. This plugin is not finished and still a work in progress!

Code: https://github.com/JakeGreer/rd-page-builder
